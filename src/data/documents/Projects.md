## 1. Field Sales Tracker
**Category:** B2B SaaS · Production App
**Year:** 2025
**Built With:** Antigravity

### The Problem
A B2B building materials manufacturer was tracking
40+ field sales reps using WhatsApp messages,
spreadsheets, and phone calls. Managers had zero
real-time visibility into where reps were, which
clients they'd visited, or what was being sold.
The tracking system was a WhatsApp group and gut feeling.

### What I Built
A production field sales tracking system where reps
log every client visit with location and time stamps.
Managers get a real-time dashboard of team activity.
Orders and interactions logged per rep, per day.
Role-based access — reps see their own data,
managers see everything.

### What It Does
- Field reps log visits, clients met, orders placed
- Managers get real-time visibility into team activity
- Location and time-stamped entries
- Dashboard with daily and weekly performance overview
- Role-based access — rep view vs manager view
- Daily reports and streak tracking
- 40+ field rep management at once
- Automated data collection and categorization

### The Result
Started with one manufacturer. Now running in
production across 10 companies with daily active
users every single day. Not a prototype — a real
business tool companies depend on.

### Tech Stack
Antigravity · Next.js · React · Supabase · PostgreSQL

### Links
Live App: salesapp.thetomorrowsteam.com
Status: Live in production — credential-based access

## 2. Deep Researcher Agent
**Category:** AI Research Tool · Multi-Agent System
**Year:** 2025
**Built With:** Antigravity
**Award:** 🏆 1st Place — Outskill AI Generalist Hackathon

### The Problem
LLMs give confident answers from stale training data.
For anything time-sensitive or nuanced, you need
something that actually goes out and reads the web —
not just recites what it memorized. Most AI tools
tell you what they already know. That's not research.
That's recall.

### What I Built
A multi-agent AI research system where specialized
sub-agents each have a defined role:
- One agent plans the research strategy
- Others execute parallel web searches
- One synthesizes findings across sources
- A final layer quality-checks the output

The result feels less like chatting with an AI and
more like having a research assistant.

### What It Does
- Takes any research query as input
- Multi-agent architecture with specialized roles
- Active web research with real-time data
- Parallel search execution for speed
- Source citation and confidence scoring
- Iterative refinement based on findings
- Hackathon-winning reliability

### The Result
Won 1st place at the Outskill AI Generalist Hackathon
competing against builders globally.

### Tech Stack
Antigravity · Python · LangChain · LangGraph ·
Streamlit · OpenAI · Tavily Search

### Links
Live Demo: my-deep-researcher-agent.streamlit.app
Status: Live demo — requires API key for full functionality
Demo Video: Available on YouTube

## 3. Travel Ease
**Category:** Consumer AI · Travel Planning
**Year:** 2025
**Built With:** Lovable
**Award:** 🏆 1st Place — Outskill AI Engineering Hackathon

### The Problem
Planning a trip means 6 open tabs, manual price
comparisons, and an itinerary you're still not
sure about. Travel agents fix this but charge a
premium and aren't available at 2am.

### What I Built
An AI travel agent that handles everything in one
conversation. Tell it where you want to go, your
budget, your vibe — it does everything else.
Also built a voice agent using Vapi so users can
speak their travel preferences instead of typing.

### What It Does
- Full itinerary generation from a single prompt
- Voice agent — speak your preferences via Vapi
  instead of filling forms
- Real-time flight prices from MakeMyTrip,
  Goibibo, and Skyscanner — cheapest options surfaced
- Hotel recommendations by budget and style
- Weather forecast for exact travel dates
- What to pack based on weather
- Local transport options at destination
- Where to eat, what to order
- Currency info, exchange rates, time zone differences
- Zero agency fees, instant results

### The Result
Won 1st place at the Outskill AI Engineering Hackathon
competing against builders globally. Built during a
48-hour hackathon sprint.

### Tech Stack
Lovable · React · TypeScript · OpenAI ·
Booking.com API · OpenWeather API · Vapi

### Links
Live App: lets-travel-with-ease.lovable.app
Status: Live and public

## 5. Carpenter Rewards — Saraswati Ply House
**Category:** B2B Loyalty · Production App
**Year:** 2025
**Built With:** Antigravity

### The Problem
In building materials, carpenters decide which brand
gets installed. They walk into every home renovation
and make the call. Most manufacturers have no way
to recognise or reward that loyalty. Everyone competes
on price alone. Saraswati Ply House tracked it manually —
phone calls, handwritten registers, best guesses.

### What I Built
A points and commission system that makes loyalty
tangible. Carpenters earn points on every purchase
and redeem them for real rewards — bikes, appliances,
electronics, vouchers.

### What It Does
- Points earned automatically on every purchase
- Carpenters scan a QR code at point of sale —
  points update instantly
- SMS notifications so carpenters know their balance
  without opening an app
- Real-time commission tracking
- Tiered rewards catalog — bikes, appliances, vouchers
- Leaderboards to drive competitive behavior
- Admin dashboard for full program management

### The Result
Live in production for Saraswati Ply House and their
network of carpenters. Real business tool used daily.

### Tech Stack
Antigravity · React · Node.js · PostgreSQL ·
QR Code API · SMS Integration

### Links
App: app.saraswatiplyhouse.com
Status: Live in production — private business tool
